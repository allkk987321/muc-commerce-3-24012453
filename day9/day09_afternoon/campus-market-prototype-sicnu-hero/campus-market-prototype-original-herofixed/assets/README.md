# 校园轻集市 HTML 原型说明

本目录包含 PC 端 Web App 高保真原型的公共样式与说明。

## 技术
- HTML 独立页面
- Tailwind CSS CDN
- Vue 3 CDN
- Element Plus CDN

## 迁移建议
后续迁移到 Vue3 + Vite + TypeScript 时，可将顶部导航、信息卡片、筛选栏、详情侧栏、发布表单拆分为独立组件。

## 图片
页面使用 Unsplash 远程图片作为真实视觉素材，后续开发可替换为本地上传或对象存储资源。


## 首页 Hero 图片替换说明

首页右侧 Hero 图片已替换为四川师范大学官网“桂花盛开，香彻川师”页面中的校园图片，用于避免原 Pexels 图片链接失效。
